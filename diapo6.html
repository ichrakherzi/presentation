shopping_cart
Achat d'un domaine
Registrars : GoDaddy, Namecheap, OVH
Coût moyen : 10-15€/an
Choisissez un nom simple et mémorable
settings
Configuration DNS
Enregistrement A : IP de votre serveur
CNAME : sous-domaines (www, blog)
MX : serveurs de messagerie
schedule
Temps de propagation
1-48 heures selon les serveurs DNS
Vérifiez avec nslookup ou outils en ligne
Patience nécessaire avant activation complète
public
Votre adresse web : simple, mémorable et professionnelle
language
Nom de domaine
L'adresse unique de votre site sur Internet
dns
Serveur DNS
Traduit le nom de domaine en adresse IP
storage
Hébergeur
Stocke les fichiers de votre site web
A
Domaine → IP
Exemple: monsite.fr → 192.168.1.1
CNAME
Sous-domaine → Domaine
Exemple: www.monsite.fr → monsite.fr
MX
Email → Serveur mail
Exemple: @monsite.fr → mail.monsite.fr
